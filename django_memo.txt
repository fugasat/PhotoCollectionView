create project
django-admin startproject project

create application
python manage.py startapp webapps

run server
python manage.py runserver

DBモデル変更を”マイグレーション”の形で保存
python manage.py makemigrations webapps

マイグレーションをDBに反映
python manage.py migrate

(参考)マイグレーションの名前を引数にとってSQLを返す
python manage.py sqlmigrate webapps 0001

シェル起動(Django APIにアクセスできる)
python manage.py shell

Adminユーザ作成
python manage.py createsuperuser
